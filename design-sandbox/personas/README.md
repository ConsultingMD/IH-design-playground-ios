# Personas

Store **design-safe** persona summaries here so Cursor can read them without hunting across tools.

## How to maintain

1. Keep one short file per primary persona (see `OVERVIEW.md` for the index).
2. Mirror whatever your org considers canonical (product marketing, research repo, etc.) — this folder is a **working copy** for the sandbox.
3. Remove or anonymize anything that shouldn’t live in git.

## Using personas in chat

- Mention `@design-sandbox/personas/OVERVIEW.md` when you want every persona considered.
- Mention a specific file (e.g. `@design-sandbox/personas/persona-primary-care-seeker.md`) for targeted scenarios.
